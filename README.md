## 🎓 PrepAI — AI Interview Preparation Platform

PrepAI is a modern AI-powered interview preparation platform designed to help students prepare for technical interviews through mock interviews, resume analysis, coding practice, and AI-generated feedback.

---

### ✨ Features

- 🤖 AI-powered mock interviews
- 📄 Resume ATS analysis
- 💻 Coding challenges with evaluation
- 📊 Performance analytics dashboard
- 🔐 JWT authentication system
- 🌙 Responsive modern UI
- 📈 Real-time progress tracking
- 🧠 AI-generated coding questions

---

### 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- Zustand
- OpenAI API
